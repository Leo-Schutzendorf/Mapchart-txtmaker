How to use this in MapChart:
1. Go to mapchart.net/usa-counties.html
2. Click SAVE/LOAD MAP CONFIGURATION → Choose file → Open
3. The map will color all counties automatically.
Enter a presidential election year and click Generate.

Reassign Other Votes

Some election years have significant third-party or independent candidates (e.g. Perot 1992, Wallace 1968). Choose how those votes are counted when determining each county's winner.

This only affects which party "wins" a county — it does not alter the underlying vote totals.

Show third parties separately Counties won by a third party are coloured in their own colour (orange for Other, green for Perot, purple for Wallace, etc.). This is the default behaviour.
All other votes → Democrat Every non-Republican vote is added to the Democratic total. Useful for "what if Perot hadn't run" counterfactuals where third-party voters are assumed to have preferred the Democrat.
All other votes → Republican Every non-Democrat vote is added to the Republican total. Useful for counterfactuals where third-party voters are assumed to have preferred the Republican.
